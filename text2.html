When creating a web page, you add tags (Known as markup) to contents of the pages.
These tags provide extra meaning and allow browsers to show users the appropriate structure for the page.

  • Headings and paragraphs.
  • Bold, italic, emphasis.
  • Structural and semantica markup.
    • Stuctural-markup: the elements that you can use to describi both headings and paragraphs.
    • Semantic markup: Which provides extra information;
      such as where emphasis is placed in a sentence, that something you have written is a quotation
      (and who said it), the mening of acronyms,and so on
